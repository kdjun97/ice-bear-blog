# Virtual Key Code

win32 programming을 위한 virtual key code  
표의 값들은 16진수 값이고, [이 곳](https://docs.microsoft.com/ko-kr/windows/win32/inputdev/virtual-key-codes)에서 값을 가져왔다.  

Key | 값
--------------|-------------------------
`backspace`   | 0x08
`Tab`         | 0x09
